## 🖼️ imgBantu – AI Image Generator

**imgBantu** is a web-based **AI image generator** that uses advanced prompting and modern web technologies to turn text into stunning visuals — instantly.

Designed to be **fast**, **responsive**, and **easy to use**, imgBantu lets you create AI-generated images directly from your browser, with no setup required.

It's a creative tool built for **experimentation**, **imagination**, and showcasing the power of **prompt engineering**.
